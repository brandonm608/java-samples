# Strings

## Longest Common Substring
There are various solutions to the Longest Common Substring problem. For comparison purposes the
progression has been kept from naive dynamic programming using a two-dimensional array to more memory efficient
versions of this algorithm. There are two versions of the single array dynamic programming algorithm. A single threaded
version, which is likely all you would ever need. The other solution is multithreaded because I could and to see at
what ridiculous string lengths you would start to see any kind of gains.
